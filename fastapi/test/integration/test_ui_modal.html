🏥 세브란스 편집기 - UI 모달 테스트

📋 테스트 가이드

아래 체크리스트를 따라 UI 기능을 확인하세요:

1. 개정 업로드 모달 테스트

  • PDF와 DOCX 파일 모두 선택 필수
  • 하나만 선택 시 경고 메시지 표시
  • 모달 크기가 90% x 90%로 설정됨
  • 모든 버튼과 내용이 보임
  • 모달 내부 스크롤 시 모달이 닫히지 않음
  • 배경 클릭 시에만 모달 닫힘

2. 비교 모달 테스트

  • 파싱 완료 후 자동으로 비교 모달 열림
  • 미리보기 없이 바로 비교 화면 표시
  • PDF 결과가 왼쪽에 표시
  • DOCX 결과가 오른쪽에 표시
  • 한쪽 스크롤 시 다른 쪽도 동기화
  • 스크롤 위치가 비율적으로 매칭
  • 병합 버튼이 하단에 표시
  • 클릭 시 병합 API 호출

3. 파일 저장 테스트

  • 저장된 파일명에 YYYYMMDD_HHMMSS 형식 포함
  • PDF, DOCX, TXT, JSON, Merged 모두 적용
  • applib/pdf/ - PDF 파일
  • applib/docx/ - DOCX 파일
  • applib/text/ - 텍스트 파일
  • applib/json/ - JSON 파일
  • applib/merged/ - 병합 파일
