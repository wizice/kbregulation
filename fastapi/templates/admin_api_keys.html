API 키 사용 안내

  • API 키를 사용하면 로그인 없이도 API에 접근할 수 있습니다.
  • 생성된 키는 생성 시에만 확인 가능하며, 이후에는 다시 볼 수 없습니다.
  • 키가 유출된 경우 즉시 삭제하고 새로운 키를 생성하세요.
  • 사용자당 최대 10개의 API 키를 생성할 수 있습니다.

API 사용 예시

cURL

# API 키로 요청하기
curl -H "Authorization: Bearer kbr_live_abc123..." \
  https://{{ request.headers.get('host', 'localhost') }}/api/v1/regulations/current

Python

# Python requests 라이브러리 사용
import requests

API_KEY = "kbr_live_abc123..."
headers = {"Authorization": f"Bearer {API_KEY}"}

response = requests.get(
  "https://{{ request.headers.get('host', 'localhost') }}/api/v1/regulations/current",
  headers=headers
)
print(response.json())

JavaScript

// fetch API 사용
const API_KEY = "kbr_live_abc123...";

fetch("https://{{ request.headers.get('host', 'localhost') }}/api/v1/regulations/current", {
  headers: { "Authorization": `Bearer ${API_KEY}` }
})
.then(res => res.json())
.then(data => console.log(data));
