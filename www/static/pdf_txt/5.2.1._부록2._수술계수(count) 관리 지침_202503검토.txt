5.2.1. 부록 2. 수술계수(count) 관리 지침 담당부서 수술실 제정일 2010.02. 최종개정일 2023.12. 최종검토일 2025.03.

수술계수(Count) 관리 지침 I. 필요성 거즈, Tape, Needle, Sharps, Instrument Count는 수술팀의 의무이며 이물질이 환자의 몸에 남아 위험을 초래할 수 있는 상황을 예방하여 최적의 수술결과를 위함이다. <개정 2014.10> II. Count 품목 <신설 2014.10>

1. Tape, Gauze

2. 모든 Sharps류

3. 수술기구

4. 기타 체내에 들어갈 수 있는 모든 물품 III. Count 과정

1. 1 차 Count
1) 수술시작 전에 시행하며 순회간호사와 소독간호사가 함께 시행한다.

2) 수술 중 물품을 추가할 때도 동일한 방법으로 한다.

3) 1 차 Count 가 맞지 않는 묶음은 즉시 수술방 밖으로 내보낸다.

2. 2 차 Count
1) 체강을 닫기 직전에 하며, 소독간호사와 순회간호사가 동시에 시행한다.

2) 순서: 소독영역 (수술영역, Mayo stand, Suture table ) → 부소독 영역

3) 체강을 열지 않는 수술은 피부봉합 직전 2차 Count에서 종료한다.

3. 3 차 Count
1) 피부를 봉합하기 직전에 시행하며 체강을 여는 경우 반드시 시행한다.

2) 2 차 Count와 동일한 방법으로 시행한다.

4. Count 물품이 추가되는 경우 동시 Count 후 기록지에 기록하고 서명한다.

5. 인수인계 시
1) Count 완료 후 인수인계 한다. 불일치 시 시점이 명확하지 않는 경우 동시책임을 갖는다.

2) 수술에 따라 인수인계 시간을 배려하여 인수인계 및 Count를 충실히 할 수 있도록 한다.
Ⅳ. Count 지침 모든 Count 과정은 두 사람이 함께 소리를 내어 계수하여야 하며, 계수 확인 뿐 아니라 물품의 원형이 그대로 유지되어 있는지도 확인해야 한다. <신설 2014.10>

1. Tape, Guaze Count
1) 모든 수술에는 X-ray detectable gauze 사용을 기본으로 한다

2) Dressing 시는 Non X-ray detectable gauze를 사용하여 Count에 혼동을 주지 않는다.

3) 거즈 Count는 보면서 소리 내어 소독간호사와 순환간호사가 동시에 시행한다

4) 린넨이나 쓰레기통은 모든 Count가 완전하게 종료될 때까지 수술방 밖으로 옮기지 않는다

5) 원래의 모양 그대로 사용하며 부득이한 경우 잘라서 사용하면 나머지 거즈는 기구에 물려 Count 시 오류를 예방한다.

6) 소아 개복술 시 체강 내에 거즈를 사용하는 경우 기구에 물려서 사용한다.
이때 사용량이 많으면 Double Count 시행 후 순회간호사는 수시로 중간 정산한다.

7) 중간 정산 시 Count work sheet에 반드시 쓰레기통에 들어간 수량만큼을 제외한 기록을 남기고 서명한다. (예: 총 30 장중 10 장을 중간 정산한 경우 10/남은 수량 20 해당항목에 기록하고 서명한다).

8) 불일치 시 쓰레기통에 있는 모든 거즈를 다시 Count한다.

9) Packing gauze는 4 x 5 x 4 X-ray gauze로 하는 것을 원칙으로 한다.

10) Tape의 경우 순환간호사는 Tape의 끈이 가지런히 보이게 5장씩 나열 하여 소독간호가 쉽게 알아볼 수 있도록 한다.

2. Sharps Count
1) 모든 수술에 Needle Count를 시행한다.

2) Count work sheet의 Syringe needle 항목에는 Syringe needle, Cut down needle, Spinal needle, Dental needle을 포함한 총 수량을 기록한다.

3) 수술에 사용하는 모든 수술용 칼, 전기소작기용 tip 등의 Count를 시행한다.

3. 기구 Count
1) 수술에 사용하는 모든 기구를 Count 하고 완전성까지 확인한다. <신설 2014.10>

2) 소독간호사와 순회간호사는 소리 내어 보면서 동시에 시행한다.
* 특별한 주의 요하는 기구: Bulldog clamp, Vessel clip

4. 기타 체내에 들어갈 수 있는 모든 물품 <신설 2019.02>
1) 수술 중 사용한 체내에 들어갈 수 있는 모든 물품을 Count 하고 완전성까지 확인한다.
V. 응급수술

1. 가능한 순회간호사는 소독간호사가 환자가 도착하면 손 소독을 하도록 배려하고 타 인력의 도움을 요청하여 반드시 Double Count를 수행한다.

2. 부득이한 사유로 Count를 수행 못한 경우 Count work sheet과 수술간호기록지에 사유를 기록한다.
